<h2 id="sobre">1. Sobre o Projeto</h2>

<p>
Este projeto consiste em uma página de perfil desenvolvida utilizando Django.
O sistema foi organizado seguindo a estrutura padrão do framework,
separando templates, arquivos estáticos e scripts para facilitar manutenção.
</p>

<p>
O projeto simula uma página de apresentação pessoal com informações,
estilização customizada e interações básicas com JavaScript.
</p>

<hr>

<h2 id="objetivo">2. Objetivo do Projeto</h2>

<p>
O principal objetivo foi praticar conceitos fundamentais do Django,
incluindo:
</p>

<ul>
<li>Estrutura de templates</li>
<li>Arquivos estáticos</li>
<li>Boas práticas de organização</li>
<li>HTML semântico</li>
<li>Integração entre frontend e Django</li>
</ul>
